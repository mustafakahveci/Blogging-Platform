# Blogging Platform UI

Frontend application of the Blogging Platform project.

Built with **React**, **Vite**, and **Tailwind CSS**.  
The application communicates with the Spring Boot backend through REST APIs and provides a modern blog-style user interface.

## Features

- Authentication (Login / Register)
- Create, edit and delete posts
- Public and private profile pages
- Search and category filtering
- Pagination
- Responsive design

## Tech Stack

- React
- React Router
- Axios
- Tailwind CSS
- Vite

## Notes

This frontend was built as part of a full-stack blogging platform project to improve frontend integration skills while working primarily with a backend-focused architecture.
