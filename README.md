# Blogging Platform

A full-stack blogging platform project built with modern web technologies.

This repository contains both the backend API and the frontend application of the platform.

## Project Structure

```text
Blogging-Platform/
├── blogging-platform-api/   # Spring Boot REST API
└── blogging-platform-ui/    # Frontend application (coming soon)
```

## Backend Features

The backend application includes:

- JWT Authentication & Authorization
- User Registration & Login
- Blog Post CRUD Operations
- Ownership-based Authorization
- User Profile Management
- Pagination, Sorting & Filtering
- Category-based Search
- Swagger / OpenAPI Documentation
- Global Exception Handling
- Secure Environment Variables Configuration

## Tech Stack

### Backend

- Java 21+
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA (Hibernate)
- MySQL
- Lombok
- Swagger / OpenAPI
- Maven

### Frontend

Frontend implementation is planned and will be added in the future.
